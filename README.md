# Phan_mem_thi_trac_nghiem
Phần mềm thi trắc nghiệm bằng C# Winform <br>
Đây là một project nho nhỏ phát triển từ bài tập lớn môn lập trình C# winform <br>
Project này nhóm tụi em đang trong quá trình hoàn thiện <br>
Hiện tại project chạy local thì bind đc data với sql <br>
